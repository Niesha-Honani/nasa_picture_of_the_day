# nasa_picture_of_the_day
# REQUIREMENTS
**NASA Picture API: https://api.nasa.gov/**
 1) Create a small web app using HTML, CSS, and JavaScript
 2) Add a button that fetches and displays NASA’s Picture of the Day using the NASA API
 3) Show the image, title, and description on the page
 4) Style the app so it is clean and readable

**Stretch challenge:** allow the user to choose a date and view that day’s picture
